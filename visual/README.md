## 电商平台实时监控系统
![效果图](./public/static/img/localhost_8999_.png)  

### 技术栈  
`vue2` + `vuex` + `axios` +  `echarts`  + `koa` + `webSocket` + `nodejs`

### 安装
```
npm install
```

### 运行
```
npm run serve
```

### 打包
```
npm run build
```
